## Getting started

### Requirements
- Ruby v2.2 or higher
- Jekyll v3.5 or higher
- Bundler 1.12 or higher

### Installing
- Clone this repository and navigate into it:<br>
  `$ git clone https://github.com/mnyrop/annotate.git && cd annotate`
- Install dependencies:<br>
  `$ bundle install`
